## What is WALTA? {{ anchor: false }}

WALTA is an infrastructure platform that enables autonomous financial transactions through AI agents. Our platform provides the tools and APIs necessary to build intelligent agents that can:

- Search and discover products across multiple vendors
- Process secure payments autonomously
- Manage transactions and track their status
- Integrate with existing business systems
- Optimize financial operations through AI

## Getting Started {{ anchor: false }}

To begin building with WALTA:

1. Create an account in the [developer portal](#)
2. Generate your API keys for authentication
3. Choose your integration method:
   - Use our HTTP APIs for direct integration
   - Leverage our SDKs for your preferred programming language
   - Build custom AI agents using our agent framework
