# Quickstart

This guide will help you get started with WALTA's autonomous finance infrastructure. We'll cover how to set up your environment, create your first AI agent, and process payments through our platform. {{ className: 'lead' }}

<Note>
  Before you can use WALTA's services, you'll need to:
  1. Create an account in the [developer portal](#)
  2. Generate your API key
  3. Set up your Stripe account for payment processing
</Note>

## Set up your environment

First, let's set up your development environment. WALTA provides SDKs for Python and JavaScript, with more languages coming soon.

<CodeGroup>

```bash {{ title: 'Python' }}
# Install the WALTA Python SDK
pip install walta-api

# Install additional dependencies for AI agents
pip install langchain openai
```

```bash {{ title: 'JavaScript' }}
# Install the WALTA JavaScript SDK
npm install @walta/api

# Install additional dependencies for AI agents
npm install langchain openai
```

</CodeGroup>

## Initialize your API client

Once you have the SDK installed, you'll need to initialize your API client with your API key.

<CodeGroup>

```python
from walta_api import WaltaApi

# Initialize the API client
api = WaltaApi(api_key="your_api_key")
```

```javascript
import { WaltaApi } from '@walta/api';

// Initialize the API client
const api = new WaltaApi({
  apiKey: 'your_api_key'
});
```

</CodeGroup>

## Create your first AI agent

Now, let's create a simple AI agent that can search for products and process payments.

<CodeGroup>

```python
from langchain.agents import Tool
from walta_api import WaltaApi

# Initialize the API client
api = WaltaApi(api_key="your_api_key")

# Create tools for the agent
tools = [
    Tool(
        name="search_products",
        func=api.get_products,
        description="Search for products by various criteria"
    ),
    Tool(
        name="process_payment",
        func=api.send_payment,
        description="Process a payment for a product"
    )
]

# Create your agent using LangChain
# (Implementation details will depend on your specific agent requirements)
```

```javascript
import { WaltaApi } from '@walta/api';
import { Tool } from 'langchain/tools';

// Initialize the API client
const api = new WaltaApi({
  apiKey: 'your_api_key'
});

// Create tools for the agent
const tools = [
  new Tool({
    name: 'search_products',
    func: api.getProducts,
    description: 'Search for products by various criteria'
  }),
  new Tool({
    name: 'process_payment',
    func: api.sendPayment,
    description: 'Process a payment for a product'
  })
];

// Create your agent using LangChain
// (Implementation details will depend on your specific agent requirements)
```

</CodeGroup>

## Make your first API request

Let's search for products and process a payment using your agent.

<CodeGroup>

```python
# Search for products
products = api.get_products(
    type="service",
    price=100,
    vendor_name="ExampleVendor"
)

# Process a payment
payment = api.send_payment(
    product_id="prod_123",
    quantity=1,
    metadata={
        "order_id": "order_456"
    }
)
```

```javascript
// Search for products
const products = await api.getProducts({
  type: 'service',
  price: 100,
  vendorName: 'ExampleVendor'
});

// Process a payment
const payment = await api.sendPayment({
  productId: 'prod_123',
  quantity: 1,
  metadata: {
    order_id: 'order_456'
  }
});
```

</CodeGroup>

## Test your setup

To verify everything is working correctly, try this simple test:

<CodeGroup>

```python
# Test product search
products = api.get_products()
print(f"Found {len(products['products'])} products")

# Test vendor list
vendors = api.get_vendors()
print(f"Found {len(vendors['vendors'])} vendors")

# Test types
types = api.get_types()
print(f"Found {len(types['types'])} product types")
```

```javascript
// Test product search
const products = await api.getProducts();
console.log(`Found ${products.products.length} products`);

// Test vendor list
const vendors = await api.getVendors();
console.log(`Found ${vendors.vendors.length} vendors`);

// Test types
const types = await api.getTypes();
console.log(`Found ${types.types.length} product types`);
```

</CodeGroup>
